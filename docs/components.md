# Components Documentation

## UI Components (Shadcn UI)
Located in `components/ui`. These are accessible, reusable components built on Radix UI.
- `button.tsx`: Standard button component.
- `card.tsx`: Card container for content.
- `input.tsx`: Form input fields.
- `...`: (List will be expanded as we audit).

## Feature Components
- **Auth**: `LoginForm`, `SignupForm`.
- **Dashboard**: `Sidebar`, `Header`, `StatsCard`.
- **Feed**: `PostCard`, `CreatePost`.

## Theme
- `theme-provider.tsx`: Handles dark/light mode switching using `next-themes`.
